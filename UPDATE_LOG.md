# 更新日誌

## 2025-12-19
摘要仍保留功能變更；與打包相關的說明已移除。

- `fast_video_slice.py`：支援在時間區間前加入「標題,HH:MM:SS -> HH:MM:SS」，檔名使用標題（經檔名安全清理），並檢查重名；未提供標題時仍用 `clip_###`。
- `USAGE.md`：補充標題輸入格式、檔名規則與重名檢查說明。
- `readme.md`：規格書同步標題輸入與輸出檔名行為。
- `UI_SPEC.md`：UI 規格加入標題輸入、重名檢查與檔名邏輯。
- 提示詞樣式：提供 AI 產出工具輸入格式的區間列表模板（每行「標題,HH:MM:SS -> HH:MM:SS」，標題唯一），方便複製使用。
- 標題清理：`sanitize_title` 允許中文字，避免中文標題被判定不合法。
- 區間表格：新增「編輯」按鈕與雙擊列即可開啟對話框編輯，避免在窄格內直接改文字。
- 區間表格欄位：標題/時間/備註欄改為可手動拖曳調整寬度，預設欄寬較大並啟用平滑水平捲動。
- ffmpeg/ffprobe 搜尋與執行：支援自訂/隨包二進位，並清除干擾環境變數以減少庫衝突；GUI log 會顯示實際使用的 ffmpeg/ffprobe 路徑。

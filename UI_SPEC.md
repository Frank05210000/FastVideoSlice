# FastVideoSlice 主程式 UI/UX 規格 (草稿)

## 目標
- 以圖形介面提供影片/字幕裁切與多段輸出。
- 降低指令列操作門檻，提供清楚的錯誤提示與進度顯示。

## 主要功能
- 選擇來源影片、字幕 (SRT, UTF-8)。
- 以表格/清單管理多個時間區間（新增/刪除/複製/排序）。
- 輸出設定：輸出資料夾、檔名規則、是否檢查影片長度。
- 執行與進度：顯示 ffmpeg 進度條、目前處理的區間、log 區。
- 結果：顯示輸出檔列表、開啟輸出資料夾按鈕。

## 使用情境/流程
1. 啟動程式 → 載入上一輪設定（若有）。
2. 選擇影片與字幕檔。
3. 輸入一個或多個時間區間（支援「標題,HH:MM:SS -> HH:MM:SS」，標題用於輸出檔名且需檢查重名）。
   - 可按「新增」→ 彈出對話框輸入，或在表格內直接編輯。
4. 勾選選項：檢查影片長度（預設勾）、Verbose、檔名附加起訖時間等。
5. 按「開始裁切」→ 顯示進度條與目前區間；log 即時更新。
6. 完成後彈出結果摘要，可點開輸出資料夾。

## 介面構成 (桌面寬度)
- 頂部：檔案選擇區
  - 影片檔路徑輸入框 + 瀏覽按鈕
  - 字幕檔路徑輸入框 + 瀏覽按鈕
  - 輸出資料夾輸入框 + 瀏覽按鈕
- 中段：區間管理區
- 表格欄位：序號/標題、開始時間、結束時間、備註（可選）
- 操作按鈕：新增、刪除、複製、上移、下移
- 輸入格式提示：標題,HH:MM:SS -> HH:MM:SS（標題可留空則用序號）
- 側邊/下方：設定區
  - 勾選：檢查影片長度（預設勾）、Verbose 日誌
- 檔名選項：是否附加起訖時間片段，例如 `clip_001__00-01-10__00-01-45`；若有標題，主檔名優先使用標題（需檢查重名與非法字元清理）
- 下方：執行區
  - 「開始裁切」主要按鈕、禁用狀態切換
  - 進度條 + 目前處理的區間文字
- 底部：Log 區
  - 可清空、可另存為檔案

## 驗證與錯誤處理
- 立刻驗證：檔案存在、字幕副檔名、時間格式、start < end。
- 執行前：若啟用「檢查影片長度」，先 ffprobe；超界則彈窗 + 高亮問題列。
- 執行中：ffmpeg 失敗時顯示錯誤摘要與 log；不中斷 UI，允許重試。
- 匯入/匯出區間：支援從文字匯入（每行一段）、匯出成文字以便重複使用。

## 視覺/互動基調
- 清晰的分組與標題；表格行距適中。
- 主要操作色一色制（例如藍或橙），搭配柔和背景；錯誤用紅色高亮。
- 進度條與狀態字串同時呈現；按鈕在執行中禁用，避免重入。
- 提示/錯誤以對話框 + log 行並存，方便回顧。

## 未來擴充（預留）
- 任務佇列：一次處理多個影片/字幕組合。
- 檔名模板：自訂 `clip_{index}_{start}_{end}`。
- 簡易預覽：內建播放器跳轉到指定時間點驗證區間。
- 多語系：英文/中文介面切換。
